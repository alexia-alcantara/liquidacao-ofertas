## Instalação

Instruções para configurar o projeto local:

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/alexia-alcantara/liquidacao-ofertas

2. **Entre na pasta do projeto**:
    cd nome-do-repositorio

3. **Instale as dependências**:
    npm install

4. **Inicie o projeto**:
   npm start
